Slide com Hash

Altere o hash clicando em uma miniatura e depois atualize a página. Observe que o target não é o mesmo que o ID do elemento (veja código fonte).

	$('#slide4').slideTabs({
		seletorMiniatura: 'a',
		hash: true
	});
